##Problem Statement
-Manage users contacts list

##Requirements
Be able to:
-Add contacts
-Remove contacts
-Show contact info
-Show the phone number of a single contact
-store contacts in mutable dictionary


##Algorithm:
Inside of while True loop:
-User gets options to choose between add contacts, remove, etc.
--The selected option runs the corrosponding function
---Once function is complete, user goes back to option choice
